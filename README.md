# Пульт охраны банка

Сайт, показывающий посетителей хранилища банка

## Запуск

* Скачать код
* Установить требуемые библиотеки командой
```
python -m pip install -r requirements.txt
```
* Запустить сайт командой
```
python -m pip install -r requirements.txt
```
* Сайт открывается по адресу http://127.0.0.1:8000
